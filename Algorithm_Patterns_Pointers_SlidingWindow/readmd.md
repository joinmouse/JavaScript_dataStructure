# 双指针和滑动窗口模式总结

## 核心要点

1. **双指针一般针对有序的数组**

2. **两种指针移动模式对比**：
   - **Two Sum（对撞指针）**：指针是对撞的，一头一尾
   - **滑动窗口（同向指针）**：指针是同向的，都在左边出发，形成一个窗口

## 滑动窗口问题列表

### 1. 无重复字符的最长子串

- **文件**：[无重复字符的最长子串.js](./无重复字符的最长子串.js)
- **问题类型**：变长滑动窗口
- **窗口大小**：动态调整（遇到重复字符时收缩左边界）
- **核心思想**：使用 Set 维护窗口内无重复字符

### 2. 最大平均数 I

- **文件**：[最大平均数I.js](./最大平均数I.js)
- **问题类型**：定长滑动窗口
- **窗口大小**：固定为 k
- **核心思想**：滑动窗口时，减去移出元素，加上进入元素，更新最大和

### 3. 长度最小的子数组

- **文件**：[长度最小的子数组.js](./长度最小的子数组.js)
- **问题类型**：变长滑动窗口
- **窗口大小**：动态调整（满足条件时收缩窗口）
- **核心思想**：找到满足和 ≥ target 的最小窗口长度

---

## 变长 vs 定长滑动窗口对比

| 特性 | 变长滑动窗口 | 定长滑动窗口 |
| ---- | ----------- | ----------- |
| 窗口大小 | 动态变化 | 固定不变 |
| 典型问题 | 无重复字符的最长子串、长度最小的子数组 | 最大平均数 I |
| 窗口调整 | 根据条件收缩/扩展 | 只滑动，不改变大小 |
| 收缩时机 | 满足特定条件时收缩 | 永不收缩 |
